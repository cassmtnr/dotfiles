🏠 Dotfiles

🚀 Quick Start

git clone https://github.com/cassmtnr/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh

📦 Installation

Prerequisites

Before installing, ensure you have:

  • MacOS 10.15 or later
  • Command Line Tools: xcode-select --install
  • Git installed and configured
  • Administrative access to your machine

Automatic Installation

# One command does everything
./install.sh

The installation script will automatically:

  • Create a timestamped backup of existing configuration
  • Install Homebrew and all packages from the Brewfile
  • Install Oh My Zsh
  • Create symbolic links for all configuration files
  • Configure MacOS system defaults

🎨 Customization

⚠️ Required Setup: After installation, you'll need to configure SSH keys and Git settings for your environment.

SSH Configuration

# Generate SSH key
ssh-keygen -t ed25519 -C "your_email@example.com"

# Edit SSH config
vim ~/dotfiles/ssh/config

Git Configuration

# Set your Git details
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

🔧 Post-Installation

Node.js Setup

# Load NVM and install Node.js
node --version  # First use loads NVM
nvm install 22
nvm use 22

🚑 Troubleshooting

Common Issues

  • Slow startup: Profile with zsh -lic exit
  • SSH issues: Check keys with ssh-add -l
  • Permissions: Run chmod 600 ~/.ssh/config
