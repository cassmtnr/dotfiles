üì¶ Installation

Prerequisites

Before installing, ensure you have:

  • MacOS 10.15 or later
  • Command Line Tools: xcode-select --install
  • Git installed and configured
  • Administrative access to your machine

Automatic Installation

# One command does everything
./install.sh

The installation script will automatically:

  • Install Homebrew and all packages from the Brewfile
  • Install Oh My Zsh
  • Create symbolic links for all configuration files
  • Configure MacOS system defaults

üöë Troubleshooting

üêå Slow Shell Startup

Profile your shell to identify bottlenecks:

zsh -lic exit

NVM is lazy-loaded for faster startup

üîë SSH Connection Issues

Verify loaded SSH keys:

ssh-add -l

Keys should be in ~/.ssh/*/

üîí Permission Errors

Fix SSH config permissions:

chmod 600 ~/.ssh/config
chmod 700 ~/.ssh

SSH requires strict permissions
